- dfs를 이용하는 문제   

- 재귀를 이용하는 것보다 저장하는 객체를 만들어서 푸는 편이 더 쉽다.   

- 주어진 조건이 2개이므로, 각각의 조건을 만족하는 수를 저장한다.   

```python
import sys

a, b = map(int, sys.stdin.readline().split())

def dfs(num, target):
    temp = [(num, 1)]
    while temp:
        ans, cnt = temp.pop()

        if ans == target:
            return cnt

        if ans*2 <= target:
            temp.append((ans*2, cnt + 1))

        if int(str(ans)+"1") <= target:
            temp.append((int(str(ans)+"1"), cnt+1))

    return -1
ans = dfs(a,b)
print(ans)
```
